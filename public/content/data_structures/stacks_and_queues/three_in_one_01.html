Three in One 01

Describe how you could use a single array to implement 3 stacks. Fixed array size solution only.

See Hints

Hints

The author of the question even says in the book, that the second part of the question which you can see here is much more complicated than anything likely to be asked in a technical interview, and that the purpose is to understand the various individual parts, which one may very well be asked to do.

I had two ideas, the first was something like the suggested solution, a single array divided up, but I was unsure how to expand and contract a fixed array (required in stricter languages than java), but once I looked up the solution, an offset was the obvious solution, something I dealt with in OS class in school for navigating a FAT file system. I also had the idea of having a stack of nodes like a linked list, where each node had three data properties, one for each stack, but quickly realized why this couldn't work (stacks are not traversed, elements are entirely pushed and popped).

Solutions

The challenge in this case was understanding and translating the provided Java code into workable JavaScript.

This is the solution in full.

class FixedMultiStack {

    // constructor used to declare and set class variables
    constructor(stackSize) {
        this.numberOfStacks = 3;
        this.stackCapacity = stackSize;
        this.values = [stackSize * this.numberOfStacks];
        this.sizes = new Array(this.numberOfStacks).fill(0);
    }

    // check if full
    // otherwise push onto appropriate stack
    // offsets to designated stack
    push(stackNum, value) {
        if (this.isFull(stackNum)) {
            console.error("The stack is full and a push was attempted.");
        } else {
            this.sizes[stackNum]++;
            this.values[this.indexOfTop(stackNum)] = value;
        }
    }

    // check if empty
    // otherwise pop from appropriate stack
    // offsets to designated stack
    // sets value on stack to zero
    // decrements stack's 'top index'
    // returns value
    pop(stackNum) {
        if (this.isEmpty(stackNum)) {
            console.error("The stack is empty and a pop was attempted");
        } else {
            let topIndex = this.indexOfTop(stackNum);
            let value = this.values[topIndex]
            this.values[topIndex] = 0;
            this.sizes[stackNum]--;
            return value;
        }
    }

    // check if empty
    // otherwise return value from appropriate stack
    peek(stackNum) {
        if (this.isEmpty(stackNum)) {
        console.error("The stack is empty and a peek was attempted");
        } else {
            return this.values[this.indexOfTop(stackNum)]
        }
    }

    // checks if designated stack is empty
    isEmpty(stackNum) {
        return this.sizes[stackNum] == 0;
    }

    // checks if designated stack is full
    isFull(stackNum) {
        return this.sizes[stackNum] == this.stackCapacity;
    }

    // processes offset to top of specified stack
    // returns the offset
    indexOfTop(stackNum) {
        let offset = stackNum * this.stackCapacity;
        let size = this.sizes[stackNum];
        return offset + size - 1;
        }
    }

First is the constructor with class variables, then the push function, then pop and peek, as well as required helper functions.
