Unique Characters

Implement an algorithm to determine if a string has all unique characters. What if you cannot use additional data structures?

See Hints

Hints

This is a relatively straightforward question. There are a couple factors to condider to arrive at an optimal solution.

First thing you need to know, is how many characters you need to handle. Assuming it's a string of english words, you can assume ASCII's 128 characters or Unicode's 256 characters. In this 'fixed character count' case, an array is ideal.

If number of characters is unknown or requires to consider the over a million possible characters in Unicode, a scaling data structure like a hash map might be more appropriate. It is also necessary to consider whether we are just flagging or counting, that is, whether we are counting the occrrences of characters or the number of unique characters in a string, period.

In the former case you require a data structure of only booleans, and in the latter case you would require ints at least, which will require more space to store as the algorithm runs.
